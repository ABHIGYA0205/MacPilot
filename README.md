# MacPilot

MacPilot is an Android application that remotely controls a macOS device over a local network.

## Features

- Live system status
- Lock Mac
- Sleep
- Restart
- Shutdown
- App Launcher
- Clipboard Sync
- File Transfer
- AI Commands (planned)

## Tech Stack

### Android
- Kotlin
- Jetpack Compose
- Retrofit
- MVVM

### Server
- FastAPI
- AppleScript
- psutil
